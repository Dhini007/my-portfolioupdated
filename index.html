Featured Project

Sign Language Recognition System for Deaf & Mute
🏆 BEST INTERACTIVE PROJECT

A web-based system that converts text and voice input into Indian Sign Language (ISL). Built with Python Flask and NLP techniques, it supports Tamil-to-English translation via Google Translator API, speech recognition for voice input, and generates ISL animations using SiGML files with avatar-based rendering.
